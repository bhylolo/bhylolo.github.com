# [Node.js](https://code.visualstudio.com/docs/runtimes/nodejs)
[Node.js](https://nodejs.org/en/)是一个平台，它让JavaScript的程序更快、更有扩展性的运行在服务器端。Node.js是一个运行时环境(`runtime`)，它使用[NPM](https://www.npmjs.com/)来管理Node.js的模块。
